# isaac_backend/camera/

Kamera konumlandırma ve çerçeveleme mantığı.

| Dosya | Açıklama |
|---|---|
| `placement.py` | Kamera montaj noktası seçimi ve orbit pozisyon üretimi |
| `bounds.py` | Depo iç sınırları ve koordinat kırpma yardımcıları |
| `framing.py` | Kamera frustum'unu zemine yansıtma ve entity'lere sığdırma |
