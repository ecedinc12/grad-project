# isaac_backend/layouts/realism/

Fotogerçekçilik detay geçişleri — temel layout üstüne uygulanır.

| Dosya | Açıklama |
|---|---|
| `layers.py` | Tüm realism geçişlerini orkestre eden ana katman |
| `atmosphere.py` | Atmosferik dağınıklık ve tavan detayları |
| `floor.py` | Zemin dağınıklığı ve koridor aşınması |
| `walls.py` | Duvar armatürleri ve panel dikiş detayları |
| `equipment.py` | Statik ekipman: şarj istasyonları, park halindeki forkliftler |
| `wear.py` | Aşınma ve insan müdahalesi izleri |
