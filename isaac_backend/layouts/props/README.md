# isaac_backend/layouts/props/

Depo prop'larını domain'e göre gruplandıran spawner kütüphanesi.

| Dosya | Açıklama |
|---|---|
| `ceiling.py` | Tavan armatürleri: aydınlatma, sprinkler, boru hatları |
| `dock.py` | Rampa ekipmanı: kapılar, dock leveler, kamyon |
| `equipment.py` | Operasyonel ekipman: bolard, kutu, konveyör, yapılar |
| `floor_markings.py` | Zemin işaretleri: oklar, boya kodları, tehlike taramaları |
| `wall_fixtures.py` | Duvar armatürleri: tabelalar, yangın söndürücüler, pano detayları |
